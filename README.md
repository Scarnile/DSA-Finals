# Java Version
This uses JDK Version 21-0.11. Go to JDK 21 and choose between Linux, macOS, and Windows and download the installer.
https://www.oracle.com/asean/java/technologies/downloads/#java21

If you're still getting errors go to the View > Tool Windows > Maven and then "Sync all Maven Projects" or check if the
correct java version is installed in your device by typing `java --version` on the terminal and that
java 21.0.11 appears.
